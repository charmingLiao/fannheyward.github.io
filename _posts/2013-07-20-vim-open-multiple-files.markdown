date: 2013-07-20 10:56
categories: [Dev]

```
vim -p file1 file2 ...
vim -o file1 file2 ...
vim -O file1 file2 ...
```

其中 `-p` 是在 tab 打开，`-o` 是在上下 split 打开，`-O` 是在左右横向 split 打开。via [Vim open multiple files][1].
